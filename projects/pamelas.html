Web Dev: Pamela's Diner

Date: March - April 2022

Back

Overview

For my Spring 2022 information systems class we were tasked with resdesigning the website for Pamela's Diner. This diner is a pittsbrugh staple with absolutely delicious food however their website wasn't the most intuitive or aesthetically pleasing. This was my first time using HTML, CSS, and Javascript and this is the project that initialized my interest in Front End Web Development.

Design Choices

We had some requirements that we had to fullfill in terms of content we included but overall I had a lot of creative liberty to design and experiment with whatever I want. Before starting I decided on a color scheme that matched the existing logo fairly closely. Furthermore, I had experiemented with topbar navigation during the web development labs during this class so I elected to use a sticky sidebar navigation becuase I really like the look of sidebars and wanted to try something new.

Furthermore, I liked the look of the circles and I felt circles matches the overall light, pastel theme I was going for. Due to this I elected to have my maps and menu items shown in circles. Finally, simply for the sake of learning more about them I included accordian, tab, and light box javascript elements from Jquery to better understand how javascript works and how to implement existing javascript libraries.

Fig 1. Color Scheme for Pamela's Website

Things I would Change

As this was my first time doing anything web development related it isn't nearly as accessible or aethestically pleasing as this website for example. However, I believe it is important to show where I started to demonstrate how far my web development and design skills have come in the past year.

If I were to go back now there are many changes I would make given my knowledge now. I would firstly make the page more accessible through tabbing and make it adjust dynamically for different screen sizes. Furthermore, at the time I had a very basic understanding of html and css so I would use margins to get elements where I wanted which I now know is horrible because as soon as it is a different screen from mine it will be a completely different layout.
